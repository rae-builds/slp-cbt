# SLP-CBT Project Progress

## 🏁 작업 완료 사항

### 1. 국시 환경 UI (High Fidelity)
- **헤더**: 시험명, 수험번호, 성명, 남은 시간(타이머), 글자 크기 조절(80%, 100%, 125%), 레이아웃 전환 버튼.
- **2분할 화면**: 왼쪽에는 문제 영역, 오른쪽에는 답안지(OMR) 영역이 고정되어 있어 실시간 확인이 가능합니다.
- **글자 크기 조절**: 상단 아이콘 클릭 시 **문제 영역**의 글자 크기만 개별적으로 조절됩니다. (헤더, 푸터, 답안지 제외)
- **하단 바**: 이전/다음 문제 이동, 전체 문제 대비 푼 문제 수/안 푼 문제 수 표시, '시험 종료' 버튼.

### 2. 문제 풀이 시스템
- **문제 은행**: `mockQuestions.ts`를 통한 데이터 관리.
- **답안 마킹**: 보기 선택 시 실시간 OMR 반영.
- **오답 체크**: '오답' 버튼을 통한 선택적 마킹 지원.

### 3. 부가 기능
- **형광펜**: 문제 텍스트 드래그 시 형광펜 효과 (켜기/끄기 지원).
- **메모**: 문제별 개별 메모 작성 및 저장.

### 4. 멀티미디어 문제 UI 개선
- **타이트한 테두리**: 이미지와 영상에 불필요한 여백을 제거하고 딱 맞는 검정색 테두리를 적용했습니다. (2~4페이지)
- **레이블 유지**: 이미지 하단의 `[사진1]` 텍스트를 정렬에 맞춰 유지했습니다.

### 5. 도구 및 편의 기능
- **실제 작동 계산기**: 하단 [계산기] 버튼 클릭 시 왼쪽 하단에 계산기 팝업이 나타나며, 실제 사칙연산이 가능합니다.
- **고성능 그림판**: 하단 [그림판] 버튼 클릭 시 중앙에 캔버스 모달이 나타납니다. 부드러운 드로잉, 정교한 지우개 기능을 제공하며, 문제 이동이나 창 닫기 시에도 그림 데이터가 전역 상태에 보존됩니다.
- **문제보기 모달**: 푸터의 [전체문제], [체크문제], [안푼문제] 버튼과 연동되어 각 필터링된 문항 리스트를 실시간 카운트와 함께 보여줍니다.

### 6. 시험 종료 및 결과
- 종료 버튼 클릭 시, 수험자 정보와 함께 최종 제출 현황을 보여주는 요약 화면으로 전환됩니다.
